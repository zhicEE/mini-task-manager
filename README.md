# Mini Task Manager

A simple task management web application built with Flask.

## Tech Stack

- Python
- Flask
- HTML
- Git

## Current Features

- Display tasks
- Render HTML templates

## Learning Goals

This project is built to practice:

- Flask routing
- Templates
- Python backend workflow
- Git version control

## Future Features

- Add tasks
- Set deadlines
- Mark tasks as completed
- Delete tasks
- Store data with SQLite
